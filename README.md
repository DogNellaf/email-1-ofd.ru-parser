# Парсер электронных чеков от 1-ofd.ru

Скрипт подключается к почтовому ящику через IMAP, находит письма от адреса `1-ofd.ru`, извлекает из HTML-части электронные чеки, парсит товары и сохраняет всё в удобную таблицу Excel.

Поддерживаются чеки, отправляемые сервисом **1-OF.D** (обычно приходят после покупок с онлайн-касс, подключённых к этому ОФД).

## Возможности

- Автоматический поиск всех писем от `1-ofd.ru`
- Парсинг HTML-версии письма
- Извлечение даты и времени чека
- Извлечение списка товаров (наименование, количество, цена за единицу, сумма позиции)
- Сохранение результата в Excel-файл с сортировкой по дате
- Чтение настроек из файла `.env` (безопасно хранить пароль)
